# 跨校生選課系統 Cross Course Application System

該專案主要為因應學生有 **跨校選課** 之需求所設計，為大一網頁前端課程時運用HTML、CSS所撰寫

使用者主要分為學生與審核人員兩個部分


## 功能

學生

* 登入/註冊
* 選課
* 查詢

審核人員

* 登入/註冊
* 名單統計
* 審核
* 加退選資料
